保健品

營養補充品、保健食品的成分分析與使用指南

胺基酸類

蛋白質構成要素，身體機能調節

牛磺酸 (Taurine)

建議劑量：500-2000mg/日

主要功效：

  • 心血管保護：穩定心律、降血壓
  • 視網膜健康：預防黃斑部病變
  • 神經保護：改善認知功能
  • 抗氧化：清除自由基

服用時機：餐後或運動前

精氨酸 (L-Arginine)

建議劑量：3-6g/日

主要功效：

  • 血管擴張：提升一氧化氮產生
  • 運動表現：增進血流循環
  • 免疫調節：促進T細胞活化
  • 傷口癒合：膠原蛋白合成

注意事項：心血管疾病患者謹慎使用

離胺酸 (L-Lysine)

建議劑量：1-3g/日

主要功效：

  • 膠原蛋白合成：與維生素C協同作用
  • 鈣質吸收：促進骨骼健康
  • 抗病毒：抑制HSV病毒複製
  • 肌肉恢復：減少運動後損傷

服用時機：空腹或睡前

酪胺酸 (L-Tyrosine)

建議劑量：500-2000mg/日

主要功效：

  • 神經傳導：多巴胺、腎上腺素前驅物
  • 認知功能：改善專注力和記憶
  • 壓力適應：調節HPA軸功能
  • 甲狀腺功能：T3、T4激素合成

注意事項：甲狀腺亢進患者避免

魚油類

Omega-3 必需脂肪酸，抗發炎護心血管

高濃度魚油 (84% Omega-3)

建議劑量：1-3g EPA+DHA/日

黃金比例：EPA:DHA = 3:2

主要功效：

  • 心血管保護：降三酸甘油脂、穩定心律
  • 抗發炎：抑制促發炎因子
  • 腦部發育：DHA為腦組織主要成分
  • 視網膜健康：維持視力功能
  • 關節保護：減少發炎反應

選購要點：TG型態吸收較佳，rTG次之

磷蝦油 (Krill Oil)

建議劑量：500-1000mg/日

特殊成分：蝦紅素 + 磷脂醯膽鹼

優勢特色：

  • 生物利用率高：磷脂型態易吸收
  • 蝦紅素加持：強效抗氧化劑
  • 無魚腥味：較不易引起胃部不適
  • 認知功能：磷脂醯膽鹼支持大腦
  • 抗老化：蝦紅素清除活性氧

注意事項：甲殼類過敏者禁用

魚油品質判斷標準

純度檢測
  • • 重金屬含量 < 0.1ppm
  • • 多氯聯苯 < 0.09ppm
  • • 過氧化價 < 5 meq/kg
濃度標準
  • • Omega-3 ≥ 80%
  • • EPA ≥ 45%
  • • DHA ≥ 25%
第三方認證
  • • IFOS五星認證
  • • Friend of the Sea
  • • MSC永續漁業

眼部保健

視力保護，預防退化性眼疾

葉黃素複合物

建議劑量：10-20mg 葉黃素 + 2-4mg 玉米黃質

最佳型態：FloraGLO® 專利游離型

主要功效：

  • 黃斑部保護：濾除藍光傷害
  • 視覺清晰度：提升對比敏感度
  • 抗氧化：清除視網膜自由基
  • 預防AMD：年齡相關性黃斑部病變

服用時機：隨餐服用增加吸收

花青素 (山桑子)

建議劑量：80-160mg (標準化25% 花青素)

優質來源：北歐野生山桑子

主要功效：

  • 微血管強化：改善眼部血液循環
  • 夜視能力：促進視紫質再生
  • 眼壓調節：預防青光眼
  • 視疲勞緩解：減少數位眼疲勞

搭配建議：與維生素C同時服用

蝦紅素 (Astaxanthin)

建議劑量：4-12mg/日

頂級來源：雨生紅球藻萃取

主要功效：

  • 超強抗氧化：效力為維生素E的500倍
  • 調節焦距：改善睫狀肌疲勞
  • 視網膜保護：穿透血視網膜屏障
  • 抗發炎：減少眼部發炎反應

注意事項：與油脂一起服用提升吸收

DHA (視網膜型)

建議劑量：500-1000mg/日

專利來源：algae DHA 或高純度魚油

主要功效：

  • 視網膜構成：佔視網膜脂肪酸60%
  • 視覺發育：支持嬰幼兒視力發展
  • 乾眼症改善：維持淚膜穩定性
  • 神經保護：預防視神經退化

族群建議：孕婦、兒童、老年人特別需要

胃部保護

胃黏膜修復，消化機能調節

L-麩醯胺酸 (L-Glutamine)

建議劑量：5-15g/日 (分次服用)

最佳時機：空腹或睡前

主要功效：

  • 腸黏膜修復：腸細胞主要能量來源
  • 腸漏症改善：強化腸道屏障功能
  • 免疫調節：支持腸道免疫系統
  • 潰瘍癒合：促進胃黏膜再生

注意事項：肝腎功能異常者謹慎使用

甘草萃取物 (DGL)

建議劑量：380-760mg DGL，餐前服用

去甘草酸：避免血壓上升副作用

主要功效：

  • 黏膜保護：刺激黏液分泌
  • 抗發炎：抑制胃部發炎反應
  • H. pylori：輔助清除幽門螺旋桿菌
  • 潰瘍治療：加速潰瘍癒合過程

服用方式：咀嚼後吞嚥效果更佳

鋅卡諾汀 (Zinc Carnosine)

建議劑量：75-150mg/日 (含鋅16-32mg)

專利形式：PepZin GI® 螯合鋅

主要功效：

  • 胃黏膜修復：促進上皮細胞再生
  • 抗氧化：清除胃部自由基
  • 抗菌作用：抑制H. pylori生長
  • 創傷癒合：加速組織修復過程

服用時機：空腹時服用，避免與鈣鐵同時

胃部益生菌

推薦菌株：Lactobacillus reuteri DSM 17938

菌數規格：100億CFU以上

主要功效：

  • 菌群平衡：抑制致病菌增殖
  • 胃酸調節：維持適當pH值
  • 免疫調節：強化腸道免疫
  • H. pylori對抗：競爭性排斥

保存條件：冷藏保存維持活性

礦物質類

必需微量元素，生理機能維持

鎂 (Magnesium)

建議劑量：300-400mg/日

最佳形式：甘氨酸鎂、檸檬酸鎂

主要功效：

  • 肌肉放鬆：預防抽筋
  • 神經傳導：維持神經功能
  • 心律調節：穩定心臟節律
  • 骨骼健康：協助鈣質吸收

服用時機：睡前，有助睡眠

鋅 (Zinc)

建議劑量：8-15mg/日

最佳形式：比糖酸鋅、檸檬酸鋅

主要功效：

  • 免疫調節：T細胞活化
  • 創傷癒合：膠原蛋白合成
  • 味覺嗅覺：感官功能維持
  • 生殖健康：精子品質提升

注意事項：避免與鈣鐵同時服用

硒 (Selenium)

建議劑量：50-200μg/日

最佳形式：硒蛋氨酸、酵母硒

主要功效：

  • 抗氧化：谷胱甘肽過氧化酶輔因子
  • 甲狀腺：T4轉T3轉換
  • 癌症預防：DNA損傷保護
  • 心血管：抗動脈硬化

注意事項：過量有毒，勿超過400μg

鉻 (Chromium)

建議劑量：50-200μg/日

最佳形式：吡啶甲酸鉻

主要功效：

  • 血糖調節：增強胰島素敏感性
  • 脂質代謝：改善膽固醇比例
  • 體重管理：減少對糖分渴望
  • 肌肉合成：協助蛋白質利用

服用時機：餐前30分鐘

碘 (Iodine)

建議劑量：150-300μg/日

最佳形式：海帶碘、碘化鉀

主要功效：

  • 甲狀腺：T3、T4激素合成
  • 新陳代謝：調節基礎代謝率
  • 神經發育：胎兒腦部發育
  • 乳腺健康：預防纖維囊腫

注意事項：甲狀腺疾病患者需醫師指導

鐵 (Iron)

建議劑量：18mg/日 (女性), 8mg/日 (男性)

最佳形式：甘氨酸亞鐵、琥珀酸亞鐵

主要功效：

  • 血紅蛋白：氧氣運輸載體
  • 能量代謝：電子傳遞鏈
  • 免疫功能：白血球活性
  • 認知功能：大腦氧合

吸收促進：與維生素C同服

維生素類

必需有機化合物，生理代謝輔酶

維生素 D3 (膽鈣化醇)

建議劑量：1000-4000 IU/日

目標血清濃度：30-50 ng/mL

主要功效：

  • 骨骼健康：促進鈣磷吸收
  • 免疫調節：調控免疫細胞
  • 肌肉功能：維持肌力
  • 心血管：血管內皮保護
  • 癌症預防：細胞分化調控

搭配建議：與維生素K2、鎂同時服用

維生素 K2 (MK-7)

建議劑量：100-200μg/日

最佳形式：MK-7 (半衰期長)

主要功效：

  • 骨質密度：活化骨鈣蛋白
  • 鈣質定向：引導鈣質入骨
  • 血管保護：抑制鈣化
  • 凝血功能：凝血因子活化

注意事項：服用抗凝血藥物者需醫師評估

維生素 C (抗壞血酸)

建議劑量：500-2000mg/日 (分次服用)

推薦形式：緩釋型、酯化維生素C

主要功效：

  • 膠原蛋白：羥化反應輔酶
  • 抗氧化：清除自由基
  • 免疫支持：白血球功能
  • 鐵質吸收：還原三價鐵
  • 神經保護：神經傳導物質合成

服用時機：餐後，減少胃部刺激

B群維生素複合物

推薦形式：活性型B群 (甲基化型態)

關鍵成分：

  • B1 (硫胺素)：能量代謝
  • B2 (核黃素)：抗氧化酵素
  • B3 (菸鹼酸)：NAD輔酶
  • B6 (吡哆醇)：胺基酸代謝
  • B12 (甲鈷胺)：神經保護
  • 葉酸 (5-MTHF)：DNA合成

服用時機：早餐後，提供全日能量

益生菌類

腸道菌群平衡，免疫機能調節

多株複合益生菌

建議菌數：100-500億CFU/日

核心菌株組合：

  • Lactobacillus acidophilus：乳酸產生
  • Bifidobacterium longum：免疫調節
  • L. rhamnosus GG：腸道屏障
  • B. breve：抗過敏
  • L. plantarum：抗發炎
  • B. lactis：消化支持

主要功效：

  • 腸道健康：維持菌群平衡
  • 免疫調節：調控免疫反應
  • 營養吸收：提升營養利用率
  • 情緒調節：腸腦軸調控

後生元 (Postbiotics)

新興概念：益生菌代謝產物

主要成分：

  • 短鏈脂肪酸 (SCFA)：丁酸、丙酸
  • 細菌素：天然抗菌物質
  • 胞壁酸：免疫調節
  • 多醣類：益生元效果

優勢特色：

  • 穩定性佳：不需冷藏保存
  • 立即見效：無需定殖時間
  • 安全性高：適合免疫力低下者
  • 品質一致：不受製程影響

益生菌選購與保存指南

品質標準
  • • 菌株標示明確 (學名+菌株號)
  • • 保證菌數到有效期限
  • • 第三方檢測認證
  • • 腸溶膠囊包覆
服用要點
  • • 空腹或餐前30分鐘
  • • 避免熱水送服
  • • 與抗生素間隔2小時
  • • 連續服用4-8週觀察效果
保存方式
  • • 冷藏保存 (2-8°C)
  • • 避免陽光直射
  • • 密封防潮
  • • 注意有效期限
