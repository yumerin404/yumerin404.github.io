# 夢鈴領域 - Yumerin's Domain

一個基於 Nuxt 3 的多元學習與創作平台，涵蓋 AI 技術、遊戲、投資、學習、醫學、小說、政治、程式設計、寫作等多個領域。

## 🌟 特色功能

- **響應式設計** - 支援桌面端與行動端
- **暗色主題** - 專業的深色介面設計
- **SEO 優化** - 完整的 SEO 設定與結構化資料
- **靜態生成** - 使用 Nuxt 3 SSG 模式，快速載入
- **無障礙設計** - 符合網頁無障礙標準

## 🛠️ 技術架構

- **框架**: Nuxt 3.17.7
- **UI 庫**: @nuxt/ui 3.2.0
- **樣式**: Tailwind CSS 4.1.11
- **圖示**: @nuxt/icon
- **部署**: GitHub Pages

## 🚀 使用方法

### 環境要求
- Node.js 16+ 
- Yarn 或 npm

### 安裝與運行

```bash
# 安裝依賴
yarn install

# 啟動開發伺服器
yarn dev

# 建置專案
yarn build
```

## 📦 部署

本專案使用 GitHub Pages 進行部署，推送程式碼後會自動建置並部署。
